Привет! Ты все еще фоткаешь экран?

Ну ничего пара шагов и ты постигнишь сриншоты - это не сложнее приготовления кофе.

Windows

Вариант первый

Сделаем скриншот с помощью комбинации трех клавиш: Win + Shift + S, экран сохранит свое состояние в момент нажатия.

Выбираем нужную область экрана.

Теперь мы сохранили снимок экрана в буфере обмена и с помощью Paste или клавиш: Ctrl + V вставить в любой чатик\текст\документ.

Mac

Вариант первый Чтобы создать снимок экрана, одновременно нажмите и удерживайте следующие три клавиши: Shift + Command + 3. Снимок экрана сохранится на рабочем столе.
Вариант второй Одновременно нажмите и удерживайте следующие три клавиши: Shift + Command + 4. Перетащите перекрестие, чтобы выбрать область экрана для создания снимка. Чтобы переместить выбранную область, нажмите и удерживайте клавишу пробела при перетаскивании. Чтобы отменить создание снимка, нажмите клавишу Esc (Escape). Чтобы создать снимок, отпустите кнопку мыши или трекпада. Снимок экрана сохранится на рабочем столе.
